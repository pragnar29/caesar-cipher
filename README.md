# Caesar Cipher Encryption and Decryption

## Overview
This project implements the Caesar Cipher algorithm in Java.

## Features
- Encrypt text using a shift value
- Decrypt encrypted text
- Supports uppercase and lowercase letters
- Preserves spaces and special characters

## Technologies Used
- Java
- Scanner Class

## How to Run

Compile:

```bash
javac CaesarCipher.java
```

Run:

```bash
java CaesarCipher
```

## Example

Input:
Message: HELLO
Shift: 3

Output:
Encrypted: KHOOR
Decrypted: HELLO
